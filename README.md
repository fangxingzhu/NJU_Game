# 🏯 鼓楼拾光
> 基于 Python + Pygame 的 2D 校园 RPG · 南京大学鼓楼校区主题  
> 漫步百年校园，拾取青春故事。

---

## ✨ 已实现功能

### 🎬 开始与角色
- **动态开始画面**：校园实景背景轮播 + 半透明遮罩 + 呼吸灯提示。
- **存档系统**：3 个独立存档槽，支持手动存档、读档、删除，自动存档可开关。
- **角色创建**：姓名输入（系统原生对话框）、性别选择、宿舍选择（滑动选取）。  
  角色立绘随性别切换。

### 🌍 大地图探索
- **地图**：40×30 图块鼓楼校区地图，包含北大楼、大礼堂、汉口路校门等 15+ 建筑。
- **玩家**：男女四方向行走动画，碰撞检测与滑动移动优化，摄像机跟随。
- **昼夜系统**：游戏内时间流逝（5 秒 = 1 分钟），白天/夜晚自动切换大地图与建筑内部背景。  
  右上角时间显示（含天数和星期）。
- **建筑交互**：靠近建筑弹出提示，按 E 进入。支持不规则建筑（多矩形 `rects`）。

### 🏛️ 建筑内部
- **通用对话**：JSON 驱动，支持多段对话与选项分支，NPC 立绘与玩家立绘同屏。
- **红色教育问答**（大礼堂）：随机抽取 5 题，答完显示成绩与评语。
- **过马路小游戏**（南园校门）：四车道避车，3 条命，胜利传送到汉口路校门，失败/退出回原位置。
- **睡觉功能**（专属宿舍）：白天睡到晚上，晚上睡到次日早晨。
- **深夜氛围**：23:00~6:00 除玩家宿舍和汉口路校门之外的建筑物显示“静悄悄”。
- **汉口路校门保安**：NPC昼夜不同对话案例。

### 🎴 彩蛋与沉浸
- **地图彩蛋**：靠近特定位置触发剧情对话与 CG 展示（永久只触发一次）。
- **音乐系统**：支持歌单自动循环播放，游戏内可一键开关音乐。

### 🕹️ 系统功能
- **游戏菜单**（ESC）：继续、状态、存档、读档、退出。
- **状态面板**：显示姓名、性别、宿舍、位置、朝向、时间等。
- **性能优化**：图片缓存、建筑触发预计算。

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Pygame 2.x
```bash
pip install pygame
```

### 运行游戏
```bash
python main.py
```

### 操作说明

| 按键 | 大地图 | 建筑内部 |
|------|--------|----------|
| 方向键 | 移动角色 | — |
| E / 空格 | 进入建筑 / 推进对话 | 推进对话 |
| Q | — | 离开建筑 |
| ESC | 打开菜单 | 打开菜单 |

## 🎮 开发规范
- **字体**：标题 `font1_path`（思源宋体），正文 `font2_path`（思源黑体）。
- **地图坐标**：`[列, 行, 宽, 高]`，行列从 0 开始，图块单位 32px。
- **建筑定义**：优先使用 `rects` 数组支持不规则形状，兼容 `rect`。
- **存档结构**：JSON 格式，包含玩家、场景、时间、宿舍、彩蛋等字段。
- **Git 提交**：功能分支开发，提交信息清晰，格式如 `feat: 功能描述`。
