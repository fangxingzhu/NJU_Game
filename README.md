# 🏛️ NJU_Game

> 基于 Python + Pygame 的 2D 校园 RPG · 南京大学鼓楼校区主题

---

## ✨ 已实现功能

### 🎬 开始与存档
- **开始画面**：品牌标题居中展示，按任意键进入存档选择。
- **存档系统**：3 个独立存档槽，支持新建、覆盖、删除、读取。
  - 记录角色数据、当前场景、游戏时间。
  - 时间状态持久化，读档恢复白天/夜晚。
  - 删除存档后新建角色自动重置时间。

### 🧑 角色创建
- **姓名**：点击输入框调用系统原生对话框（macOS / Windows / 通用 tkinter）。
- **性别**：男 / 女按钮切换，立绘背景色对应变化。
- **校验**：姓名长度限制（8 个中文或 16 个英文字符）。
- 确认后进入校园大地图。

### 🗺️ 大地图探索
- **地图**：40×30 图块地图，整张贴图渲染。
- **玩家**：
  - 四方向像素行走动画（男女不同精灵表）。
  - 基于 Tile 的碰撞检测，贴合建筑边缘。
  - 滑动移动优化，避免墙角卡死。
- **摄像机**：跟随玩家并限制在地图边界内。
- **建筑交互**：
  - 靠近建筑自动弹出 `按 E 进入` 提示。
  - 支持不规则建筑（多矩形 `rects` 定义）。
- **时间系统**：
  - 游戏内昼夜循环（现实 5 秒 = 游戏 1 分钟）。
  - **白天/夜晚大地图自动切换**
  - 右上角显示当前时间与日月图标。
- **菜单**：按 `ESC` 弹出游戏菜单（继续/状态/保存/退出）。

### 🏛️ 建筑内部
- **背景**：每栋建筑独立内部背景图，**随日夜自动切换**。
- **对话系统**：
  - 支持多段对话（数组配置），按 `空格/E` 推进。
  - 对话结束后显示提示，手动按 `Q` 离开建筑。
- **场景保留**：对话完成后不强制退出，可自由观赏内部。

## 🚀 运行方式

1. **环境要求**  
   Python 3.8+ / Pygame 2.x
   ```bash
   pip install pygame
   ```

2. **启动游戏**  
   ```bash
   python main.py
   ```

3. **操作说明**  

| 按键     | 大地图 | 建筑内部 |
|--------|-----|------|
| 方向键    | 移动角色 | —    |
| E      | 进入建筑 | —    |
| 空格 / E | —   | 推进对话 |
| Q      | —   | 离开建筑 |
| ESC    | 打开菜单 | 打开菜单 |

## 🎨 开发规范

- **字体**：标题使用 `font1_path`（思源宋体），正文/UI 使用 `font2_path`（思源黑体）。
- **地图坐标**：`[列, 行, 宽, 高]`，行列从 0 开始，图块单位 32px。
- **建筑定义**：优先使用 `rects` 数组支持不规则形状，兼容 `rect` 旧格式。
- **存档结构**：JSON 格式，包含 `player`、`scene`、`game_time` 等字段。
- **Git 提交**：小步提交，描述清晰，功能分支开发。
